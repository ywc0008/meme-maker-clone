### 화살표 함수

#### 함수 정의

화살표 함수는 함수 선언문으로 정의할 수 없고 함수 표현식으로 정의해야 한다. 호출 방식은 기존 함수와 동일하다.

    const multiply = (x, y) => x * y;
    multiply(2, 3); // 6

#### 매개변수 선언

매개변수가 여러 개인 경우 소괄호() 안에 매개변수를 선언한다.

    const arrow = (x, y) => {...};
    const arrow = x => {...};
    const arrow = () => {...};

### Array.from();

유사 배열 객체(array-like object)나 반복 가능한 객체(iterable object)를 얕게 복사해 새로운Array 객체를 만듭니다.
